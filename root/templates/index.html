The Paradox
  • Parallax design using Skrollr.js
  • Fully responsive
  • Separate css for mobile devices using enquire.js
  • Skrollr is disabled for touch devices using modernizer.js
  • Loading screen until all images are loaded using imagesloaded.js
  • Navigation pane on the right using waypoints.js and skrollr.menu.js
  • Navigation, menu, close & transitions are done with pure css using psedo elements and a lot of time :p
  • Tested on 320x480, 360x640, 768x1024, 800x1280, 1280x600, 1920x900 and 1920x1080 screens.
  • Easy to build with the right tutorials (links are on my blog).
